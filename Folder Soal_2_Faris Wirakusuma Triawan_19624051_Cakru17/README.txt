
PENJELASAN PROGRAM
------------------------------------------------------------------------
Cara Memainkannya:

open file main.exe untuk menjalankan game


Sistem Game:

Terdapat 3 Scene yaitu:

-Scene Pemilihan robot pada setiap player:
terdapat 4 robot yang berbeda yang dapat dipilih player 1 maupun player 2
cara pemilihan robot ialah dengan cara mengklik salah satu kotak pada layer
cara bergantian pemilihan player 1 dan 2 ialah mengklik bagian  tombol bulat berwarna merah

-Scene Pertarungan sederhana:
Untuk menyerang player harus mengklik tombol enter pada keyboard
player 1 akan selalu maju pertama tetapi player 2 akan mendapatkan buff tambahan health secara acak guna membuat pertarungan lebih adil
Ketika salah satu robot kalah maka akan langsung ke scene berikutnya

-Scene Ending (Ketika salah satu robot kalah):
Ketika salah satu robot kalah maka akan menuju ke scene ini.scene ini berisi text bertuliskan pemenang dan yang kalah pada battle

Di dalam kelas robot terdapat properties:
-Health
-Attack
-Warna
dan setiap robot memiliki properties yang berbeda beda

Game OOP ini menggunakan C++ dan Raylib sebagai libraries nya
